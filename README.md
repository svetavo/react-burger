### Учебный проект для работы космической бургерной

- Создание бургеров
- Работа с сервером
- Аутентификация пользователя
- Реализация отображения ленты заказов в реальном времени 

### Технологический стек:
React, Redux, TypeScript, Websocket

### Запуск приложения:
npm i

npm run dev
